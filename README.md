# Roblox Lua Injector Script

## Description
This repository contains a Lua script created for Roblox exploit injectors.  
The script is designed for in-game execution and focuses on simplicity and ease of use.

## Features
- Written in Lua
- Works in Roblox injector environments
- Simple and lightweight
- Easy to modify and expand

## Usage
1. Load the script into your Roblox Lua executor.
2. Join a game.
3. Execute the script.

## Requirements
- Roblox Lua injector / executor
- Basic Lua knowledge
